疑问？ 积化和差和和差化积需要记住吗？高考和考研要求记忆这个很好吗？
记忆方法先说下：
正、余乘系正弦，和差项“加”号连<br />
同名乘系余弦，正变余“减”号连<br />
注释：正指代正弦 正变余的减，余不变就是加号<br />
来源：完善“三角函数积化和差公式的记忆方法”阐述和差化积公式的记忆法 cnki<br />
